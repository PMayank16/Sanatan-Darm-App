# 🕉️ Sanatan Dharm (सनातन धर्म) App

A premium, two-sided spiritual marketplace built with **React Native (Expo)** and **Firebase**. This app connects Vedic Pandits with Devotees for Puja bookings, spiritual guidance, and daily rituals.

---

## ✨ Key Features

### 👥 For Devotees
- **Pandit Marketplace:** Browse and book verified Vedic Pandits based on location and rating.
- **WhatsApp-like Chat:** Real-time inquiry and messaging with Pandits.
- **Media Sharing:** Send images and voice notes for ritual guidance.
- **Mantras & Jap:** Digital Jap counter with multiple mantra options.
- **Puja Vidhi:** Step-by-step guides for various Vedic rituals.
- **Panchang:** Daily Panchang with Tithi, Nakshatra, and Muhurat.

### 🪔 For Pandits
- **Booking Dashboard:** Manage pending, approved, and inquiry requests.
- **Role-based Profile:** Specialized profile for Vedic expertise.
- **Chat & Inquiry:** Connect with devotees before booking approval.

---

## 🛠️ Tech Stack
- **Frontend:** React Native (Expo SDK 54)
- **Styling:** NativeWind (Tailwind CSS)
- **Backend:** Firebase Firestore, Authentication, & Storage
- **Navigation:** React Navigation (Stack & Tabs)
- **Localization:** i18next (Hindi & English support)

## 🚀 Getting Started

### Prerequisites
- Node.js (LTS)
- Expo Go app on your phone
- Firebase Account

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/PMayank16/Sanatan-Darm-App.git
   cd Sanatan-Darm-App
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Setup Firebase:
   - Create a project on Firebase Console.
   - Add your config to `src/config/firebase.js`.
   - Enable Firestore, Auth, and Storage.

4. Run the app:
   ```bash
   npx expo start
   ```
